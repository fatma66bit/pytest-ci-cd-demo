# Projet de démonstration CI/CD avec pytest et GitHub Actions

## Description
Ce projet démontre l'implémentation d'un pipeline CI/CD complet avec pytest et GitHub Actions.

## Fonctionnalités testées
  - Tests paramétrés
  - Fixtures avancées
  - Mocking des dépendances
  - Tests d'exceptions
  - Markers et sélection de tests
  - Rapports de couverture
